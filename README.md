## Getting Started


### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/f418me/LNbitsVoucherGenerator.git
   ```
2. Install Pyhton Packages under Prerequisites
   ```sh
   pip install pandas
   pip install PIL
   ```
<p align="right">(<a href="#readme-top">back to top</a>)</p>
